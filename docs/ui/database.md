# Module: `src/database.py`

## Overview

Backward-compatible shim module that re-exports all database engine, session factory, initialization function, and model classes from the new modular structure (`src.core.db` and `src.models.schema`). This preserves import compatibility for legacy code while the codebase migrates to a decoupled architecture.
